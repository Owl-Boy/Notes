# Simply Connected
```ad-note
title:
A topological space $X$ is **simply connected** if $X$ is path connected and for every $x_0 \in X$, $\Pi_1(X,x_0) = {1}$.
```

---
# Examples:
1. Convex subsets of $\mathbb{R} ^{n}$
2. $S ^{n}$ for $n \ge 2$.
