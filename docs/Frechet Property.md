# Frechet Property
### Definition
A Topological Space $X$ is said to be $T_1$ or **Frechet** if for all pairs of points $x, y$ there exists a pair of open sets $U, V$ such that $U$ contains $x$ but not $y$ and $V$ contains $y$ but not $x$.
