# Second Countability
```ad-note
title:
A topo space $X$ is called second countable if $\exists$ a ctble basis $\mathcal{B}$ on X that generates $\tau_X$.
```
---
# Related Results
1) Regular + 2nd Countable $\implies$ Normal
2) Metrizable + Separable $\implies$ 2nd Countable
3) Metrizable + Lindelof $\implies$ 2nd Countable
4) 2nd countable $\implies$ Lindelof
