# R_K
```ad-note
title:
Topology on $\mathbb{R}$ generated by the basis $(a,b)$ and $(a,b) \setminus K$, where $K = \{1/n : n \in \mathbb{N}\}$
```


---
# Properties
1. Hausdorff
2. Not Regular.
   Note that $K$ is closed in $\mathbb{R}_{K}$, we can't separate $0$ and $K$.
