# Integrality Gap

$OPT(LP)\le OPT(ILP)\le$ our solution

**Integrality gap** $=\max_\limits{\text{all instances}}\frac{OPT(ILP)}{OPT(LP)}$
integrality gap $\le$ approximation ratio
